class saga.source.ga_source.GASource(ga_client: object, view_id: str, extractor_name: str, data_source_name: Optional[str] = None, bulk: bool = False, **extractor_args: dict)¶

Bases: saga.source.source.Source

Class to extract data from Google Analytics API

ga_client¶

Google Analytics API client, generated with the google api SDK.

Type

object

view_id¶

GA view id, located on the admin console.

Type

str

extractor_name¶

Name of the data extractor to be used. Must be one of the names in the extractor factory.

Type

str

data_source_name¶

If provided, changes the “data_source” key from the default extractor name to the name provided, changing the write location of the results.

Type

str

bulk¶

Extract data in batches if True and single records if False. Defaults to False.

Type

bool, optional

\*\*extractor_args

Extractor keyword arguments to be used.

Type

dict

produce()¶

Yields the extracted data from the API

Yields

dict –

Dictionary containing the data and additional metadata.

Data from the API is contained in key “data”, name of the source API is in “source_api” key, name of the API data source that is being extracted is in key “data_source” and the date that the record was extracted and produced is in “produce_date” key.
