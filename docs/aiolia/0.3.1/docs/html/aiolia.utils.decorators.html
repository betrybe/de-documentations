aiolia.utils.decorators package¶

Submodules¶

class aiolia.utils.decorators.container.ContainerDecorators¶

Bases: object

Holds the decorator logic for execution profiling.

Decorator functions to profile the execution time and execution status of the components. Depends upon DataContainer and StatusContainer classes and any class that derives from the Component interface.

static error_handling(function)¶

Logs the error generated by the function.

Gets the exception thrown and save its details on the StatusContainer of the Component derived class that is executing the function.

static execution_time(function)¶

Times the function execution.

Holds start date, end date and execution duration. At the end of the decorated function, saves these three metrics on the StatusContainer object of the Component derived class and in the StatusContainer of the DataContainer that is being passed.
