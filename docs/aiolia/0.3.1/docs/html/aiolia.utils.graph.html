class aiolia.utils.graph.labeled_graph.LabeledGraph(spark_session: pyspark.sql.session.SparkSession, sources: List[dict], transforms: List[dict], sinks: List[dict])¶

Bases: object

Holds the logic of automatic DAG creation using labeled data.

Gets the Sources, Transforms and Sinks and created the desired DAG using the keys passed by the user.

It’s important to notice that this class holds the logic to create the directed graph for the pipeline, already instantiating the classes on each node.

Providing no Sources or no Sinks yields an invalid graph. Passing a cyclic structure yields an invalid graph.

Parameters

  • spark_session (SparkSession) – Spark session object provided by the user.

  • sources (List[dict]) – List of Source objects.

  • transforms (List[dict]) – List of Transform objects.

  • sinks (List[dict]) – list of Sink objects.

Raises

  • ValueError – if the sources or sinks lists are empty.

  • ValueError – if the created graph contains a cycle (therefore, not a DAG).

  • ValueError – if an incorrect direction is detected. An incorrect direction can be: transform to a source; sink to a source; sink to a transform.

class aiolia.utils.graph.unlabeled_graph.UnlabeledGraph(spark_session: pyspark.sql.session.SparkSession, sources: List[dict], transforms: List[dict], sinks: List[dict])¶

Bases: object

Holds the logic of automatically creating a directed graph based on inputs.

Gets the Sources, Transforms and Sinks inputs and map them into a graph. Since no explicit direction is provided, the graph is created following this logic: each Source input is directed to the first transform, if no transform is provided, then each Source is directed to each Sink. If Transforms are provided, each transform is directed to the next one in the provided order and the last one directs to each Sink. It’s important to notice that this class holds the logic to create the directed graph for the pipeline, already instantiating the classes on each node. Providing no Sources or no Sinks yields and invalid graph.

Parameters

  • spark_session (SparkSession) – Spark session object provided by the user.

  • sources (List[dict]) – List of Source objects.

  • transforms (List[dict]) – List of Transform objects.

  • sinks (List[dict]) – list of Sink objects.

Raises

ValueError – if the sources or sinks lists are empty.
