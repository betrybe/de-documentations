class aiolia.components.status_container.StatusContainer(component_pipeline_name: Optional[str] = None, component_class_name: Optional[str] = None, start_time: Optional[datetime.datetime] = None, end_time: Optional[datetime.datetime] = None, execution_time: Optional[datetime.datetime] = None, status: aiolia.utils.constants.component_status.ComponentExecutionStatus = <ComponentExecutionStatus.NOT_EXECUTED: 1>, error_type: Optional[Exception] = None, error_message: Optional[str] = None, error_stack_trace: Optional[str] = None)¶

Bases: object

Container for execution status.

Keeps the record of the Component execution status, including possible errors and failures. Used by the Component derived classes (such as JsonSource) to keep track of the execution status and used by the DataContainer class to keep track of its transformation roadmap, indicating by which Component it already passed and what happened at each of them.

component_pipeline_name¶

it is a generic description of what the component does and it doesn’t need to be equal or related to the class name.
